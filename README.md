# CyberFlash

Professional cross-platform Android ROM flashing tool built with Python + PySide6.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
python -m cyberflash
```

## Development

```bash
ruff check src/    # lint
pytest tests/      # run tests
```
